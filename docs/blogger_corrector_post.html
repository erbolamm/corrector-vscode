Corrector: Tu asistente de escritura en Copilot Chat

Extensión de VS Code que corrige la ortografía en español directamente dentro del chat de GitHub Copilot. Diseñada para personas con dislexia.

100% Offline · Sin dependencias · Open Source
❤️ Apoya el proyecto: 💳 PayPal ☕ Ko-fi ⭐ GitHub

¿Tienes dislexia y usas Copilot en VS Code? ¿Escribes rápido y se te escapan errores ortográficos? Con @corrector tus mensajes se corrigen automáticamente antes de llegar a Copilot. 150+ reglas de español, sin conexión a internet, sin enviar tus datos a ningún sitio.

💬 Chat de Copilot
Tú escribes:
@corrector ola vuenos dias kiero aser una app para el movil
@corrector responde:
📝 Texto corregido (6 correcciones):
> Hola buenos días quiero hacer una app para el móvil
• ola → hola (h omitida)
• vuenos → buenos (v→b)
• dias → días (tilde)
• kiero → quiero (k→qu)
• aser → hacer (h omitida)
• movil → móvil (tilde)

¿Qué hace @corrector?

🔤
150+ reglas de español B/V, H omitida, C/S/Z, G/J, LL/Y, tildes, abreviaturas de chat y mucho más.
✈️
100% Offline Todo funciona en tu máquina. Sin APIs, sin enviar tus textos a ningún servidor externo.
🚀
3 botones inteligentes Copiar al portapapeles, Enviar directo a Copilot, o Permitir siempre esas palabras.
📖
Diccionario personal Añade tus propias palabras y reglas. Se guardan entre sesiones automáticamente.
💬
Texto correcto = pasa directo Si no hay errores, tu mensaje se envía automáticamente a Copilot. Sin interrupciones.
♿
Diseñado para dislexia Pensado desde cero para personas con dificultades de escritura. Sin juicios, solo soluciones.

🛠️ Cómo instalar

1
Abre VS Code con GitHub Copilot activo
2
Busca "Corrector" en el Marketplace de extensiones (Ctrl+Shift+X)
3
Instala la extensión y abre el chat de Copilot
4
Escribe @corrector seguido de tu texto. ¡Listo!

Tipos de errores que corrige

B/V vueno → bueno, bamos → vamos
H omitida ola → hola, aser → hacer
Tildes dias → días, tambien → también
Chat/SMS q → que, xq → porque, tb → también
C/S/Z desir → decir, grasias → gracias
Y más... G/J, LL/Y, N→M antes de B/P, transposiciones

⌨️ Comandos disponibles

@corrector tu texto aquí — Corrige el texto
@corrector /ayuda — Muestra la ayuda
@corrector /agregar errorr=correcto — Añade regla personalizada
@corrector /ignorar palabra — No corregir esta palabra
@corrector /stats — Estadísticas de uso
